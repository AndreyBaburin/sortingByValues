Для тестирования запросов использовалось приложение Postman

Запуск и формат параметров:

1.Для получения всех значений (GET запрос)
Запрос: URL: http://localhost:8080/message
Входные параметры: отсутствуют
Выходные параметры: список пар ключ-значение
Список отсортирован по убыванию количества значений.
Пример ответа: [{"a":5},{"c":4},{"b":1}]
Статус: 200 OK

2.Для добавления значения (POST запрос)
Запрос: URL: http://localhost:8080/message
Входные параметры: Входные параметры: строка в теле запроса (Body) формат: Text
Пример запроса: aaaaabcccc
Выходные параметры: не возвращается никакого значения.
Статус: 201 Created

3.Для обновления значения (PUT запрос)
Запрос: URL: http://localhost:8080/message
Входные параметры: Входные параметры: строка в теле запроса (Body) формат: Text
Пример запроса: hhhelllloo
Выходные параметры: не возвращается никакого значения.
Статус: 200 OK

4.Для удаления значения (DELETE запрос)
Запрос: URL: http://localhost:8080/message
Входные параметры: отсутствуют
Выходные параметры: отсутствуют
Статус: 200 OK
